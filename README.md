## 🧠 Project Description

**Collection Manipulator** is a Python-based command-line application designed to efficiently manage student records using collection data structures such as lists and dictionaries. The system simulates how structured data is stored, accessed, and manipulated in real-world applications.

The application provides a menu-driven interface that allows users to perform multiple operations including adding, viewing, updating, and deleting student records. Each student is stored as a dictionary containing attributes such as ID, name, age, and subjects, while all records are maintained within a list acting as a dynamic database.

The program demonstrates how data collections can be used to build scalable and maintainable systems. It uses control flow structures, loops, and conditional logic to ensure smooth interaction and proper data handling. Users can also view stored subjects, showcasing data extraction from nested structures.

Although simple in design, the system reflects core concepts used in larger applications such as database management systems and backend services. It mimics how systems process input, update records, and maintain data integrity.

From an AI/ML perspective, this project builds foundational thinking by structuring data and applying logical operations—key skills required before working with datasets, preprocessing, and machine learning models.

Overall, Collection Manipulator highlights strong fundamentals in Python programming, data handling, and system design, making it a valuable stepping stone toward advanced AI and software development projects.
